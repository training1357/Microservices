==== Install Docker for Ubuntu

01. Install Docker for Ubuntu, menggunakan dokumen "Instll Docker on Ubuntu.txt"

02. Verify docker installed correctly

	$ sudo docker version
	$ sudo docker info
	
		WARNING: No swap limit support
	
03. Menghilangkan warning "No swap limit support"	

	$ sudo nano /etc/default/grub

		GRUB_CMDLINE_LINUX="cgroup_enable=memory swapaccount=1"

	$ more /etc/default/grub
	$ sudo update-grub
	$ sudo reboot
	$ sudo docker info

04. List docker command

	$ sudo docker

05. Test Docker

	$ sudo docker run hello-world

06. Run docker without sudo

	$ sudo gpasswd -a $USER docker
	
	Logout -> Login
	
	$ docker version
	$ docker info
